# 💰 Expense Splitter Pro

A Python-based desktop application to manage and split group expenses efficiently. The application calculates each person's share and minimizes the number of transactions using an optimized algorithm.

## 🚀 Features

- Add and manage participants  
- Record expenses with payer details  
- Automatic equal expense splitting  
- Optimized settlement using greedy algorithm  
- Displays "who pays whom" clearly  
- Save and load expense data using JSON  
- User-friendly GUI built with Tkinter  

## 🛠 Tech Stack

- **Programming Language:** Python  
- **GUI Framework:** Tkinter  
- **Data Storage:** JSON  
- **Concepts Used:** OOP, Algorithms, Modular Programming
